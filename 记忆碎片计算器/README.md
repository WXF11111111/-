# 卡厄斯工坊 - 记忆碎片计算器

一款专为《卡厄斯：记忆碎片》游戏设计的角色碎片评分工具，帮助玩家快速计算记忆碎片的评分和毕业率。

## 🎮 功能特点

### 角色管理
- 支持游戏中全部31个角色
- 可创建多个同名角色（区分不同自我意识等级）
- 本地存储，数据安全不丢失

### 碎片评分系统
- 基于角色类型（输出/防御/辅助）的差异化评分权重
- 自动计算6个碎片的综合评分
- 实时显示毕业率和评级（普通/A/S/SS/SSS）

### 评分算法
- 主属性满分40分，副属性每项15分
- 根据角色类型调整属性权重：
  - **输出型**：暴击、暴伤权重更高
  - **防御型**：防御、HP权重更高  
  - **辅助型**：治疗、护盾权重更高

## 🚀 使用方法

1. **打开首页**：访问 `index.html`
2. **新建角色**：点击「角色管理」→「新建角色」
3. **录入碎片**：选择角色 → 点击「录入碎片数据」
4. **选择部位**：选择碎片部位（冲击/压抑/否定/理想/渴望/想像）
5. **输入属性**：选择强化等级、主属性和副词条
6. **计算评分**：点击「计算并保存」
7. **汇总评分**：录入6个碎片后点击「汇总计算」
